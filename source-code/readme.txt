- Với các chức năng liên quan tới Service: (Tìm trên thanh tìm kiếm):
	+ View Advanced System Settings
	+ Enviroment Variables
	+ Khung dưới -> Kéo tìm Path
	+ Add: <YOUR_PATH>\WindowsPowerShell\v1.0

- Một số chức năng bên server có sử dung đường dẫn TUYỆT ĐỐI, vui lòng thay đổi để phù hợp với từng máy.
